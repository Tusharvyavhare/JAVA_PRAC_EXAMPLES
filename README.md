# ☕ Java Basic Programs

A collection of beginner-friendly Java programs that cover the fundamental concepts of Java programming. This repository is designed for students, beginners, and anyone who wants to practice basic Java programming through simple examples.

## 📚 Topics Covered

This repository includes programs such as:

* 🔄 Swap Two Numbers
* ➕ Addition, Subtraction, Multiplication, Division
* 📐 Area of Circle, Rectangle, Triangle
* 📏 Perimeter Calculations
* 🔢 Even or Odd Number
* 🔍 Largest of Three Numbers
* 🌡️ Celsius to Fahrenheit Conversion
* 🔁 Reverse a Number
* 🔢 Sum of Digits
* 🔄 Palindrome Number
* ⭐ Armstrong Number
* 🧮 Factorial of a Number
* 🌀 Fibonacci Series
* 🔢 Prime Number Check
* 📊 Greatest Common Divisor (GCD)
* ✖️ Least Common Multiple (LCM)
* 💯 Percentage Calculation
* 📥 User Input using `Scanner`
* 🖥️ Basic Output Programs
* And many more...

## 🎯 Purpose

The goal of this repository is to:

* Learn Java programming fundamentals.
* Practice basic programming logic.
* Understand syntax and control structures.
* Build a strong foundation before moving to advanced Java concepts.

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/Java-Basic-Programs.git
```

2. Open the project in your favorite Java IDE.

3. Compile the Java file.

```bash
javac FileName.java
```

4. Run the program.

```bash
java FileName
```
